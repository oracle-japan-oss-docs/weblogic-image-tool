イメージ・ツール

    イメージの作成

    createコマンドは、新しいコンテナ・イメージを作成し、リクエストされたJavaおよびWebLogicソフトウェアをインストールします。

    補助イメージの作成

    createAuxImageコマンドは、WDTインストールおよびオプションのモデル、アーカイブおよび変数ファイルを使用して新しいコンテナ・イメージを作成します。

    イメージのリベース

    rebaseコマンドは、既存のイメージから既存のWebLogicドメインを使用して新しいコンテナ・イメージを作成します。

    イメージの更新

    updateコマンドは、WebLogicパッチを既存のイメージに適用することで、新しいコンテナ・イメージを作成します。

    イメージの検査

    inspectコマンドは、コンテナ・イメージの内容をレポートします。

    キャッシュ

    Image Toolは、パッチおよびインストーラのローカル・ファイル・システムのメタデータを保持します。 cacheコマンドを使用してローカル・メタデータを操作できます。
