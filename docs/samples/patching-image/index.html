既存のイメージにパッチを適用

この例では、WebLogic Serverパッチを既存のイメージに適用する方法を示します。 イメージ・ツールを使用してパッチをダウンロードするか、手動でダウンロードできます。

ステップ

  1. 「クイック・スタート」ガイドの説明に従って、イメージを作成します。

  2. WebLogicパッチごとに、Oracle Supportからダウンロードしてキャッシュを設定します。

    たとえば、WebLogic Serverバージョン12.2.1.3.0のパッチ番号27342434をダウンロードするには:

    $ imagetool cache addPatch --patchId 27342434_12.2.1.3.0 --path /home/acmeuser/cache/p27342434_122130_Generic.zip

    ノート: patchIdの書式については、「キャッシュ」コマンドを参照してください。

  3. imagetool updateコマンドを使用してイメージを更新します:

    $ imagetool update --fromImage wls:12.2.1.3.0 --tag wls:12.2.1.3.4 --patches 27342434
