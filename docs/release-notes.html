リリース・ノート

  • 今後、Oracle Supportの推奨パッチ・リストを変更するには、--recommendedPatchesまたは--latestPSUフラグを使用するには、WebLogic Image Tool 1.9.11が必要です。
  • Oracleのセキュリティが変更されたため、イメージ・ツールの古いバージョンはパッチをダウンロードできなくなります。 WebLogic Image Tool 1.9.10以降を使用してください。
